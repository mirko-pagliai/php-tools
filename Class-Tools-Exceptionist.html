__callStatic()
__callStatic(string $name, array $arguments)

__callStatic() magic method. This method allows you to use any PHP function, using the name in "CamelCase". For example:

Exceptionist::inArray(['a', ['a', 'b', 'c']], 'Value is not in array', \LogicException::class);

This will verify the function:

in_array('a', ['a', 'b', 'c']);

And it will throw a LogicException exception with the passed message, if the function returns false.

Another example:

Exceptionist::isDir('/my/dir/path', 'This is not a directory', \RuntimeException::class);
